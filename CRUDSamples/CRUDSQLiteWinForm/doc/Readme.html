補充:
SQLite 歷史悠久, 從2000年開放至今, 非常適用於本機資料庫的需求應用, 執行效能高. 讀取資料可以多工執行, 但是寫入資料庫時, 則需依序鎖定執行.
SQLite 從 v 1.0.112 開始, 已不支援 Visual Studio 2015 從 GitHub 直接下載的方式.
以本程式開發環境為X86為例, 若仍要以 Visual Studio 2015 開發 SQLite 的程式, 則必須到 SQLite Download, 或 附檔 下載所需的檔案 sqlite-netFx46-binary-bundle-Win32-2015-1.0.112.0.zip, 自行加入 reference 中.
Visual Studio 2015 已漸漸被忽略, 該是放棄使用的時候了 !
